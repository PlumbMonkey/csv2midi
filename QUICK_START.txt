## Key Features

- 8 MIDI event types (note on/off, program change, CC, tempo, track name, etc.)
- 3 time units (ticks, beats, milliseconds)
- 2 tempo units (BPM, USPB)
- Multi-track support
- Full validation with helpful error messages
- Standards-compliant MIDI output
